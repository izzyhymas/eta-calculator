# ETA Calculator

ETA Calculator using TypeScript and JavaScript!

## Run Project Locally
1. Clone eta-calculator Repository

2. Add Vite:<br/> npm create vite@latest

3. CD Into Repository:</br> cd eta-calculator

4. npm install

5. npm run dev

You should have access to the app now!
